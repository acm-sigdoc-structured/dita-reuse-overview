# These DITA source files support the ACM SIGDOC Structured Authoring resource "How does DITA support content reuse?".

To test basic examples of DITA reuse using library topics and direct (URL) references, open map_reuse.dita in your DITA editor.

To test that same library with indirect (key-based) references, open map_reuse_with-keys.dita in your DITA editor.
